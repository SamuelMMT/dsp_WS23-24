2 Diskrete Faltung

In diesem Versuchsteil definieren wir die Impulsantwort $h[n]$ eines zeitdiskreten Filters ausgehend von einem analogen Rechteckpuls $h(t) = \text{rect} (t/\Delta T)$ mit $\Delta T = 250\,\mu$s, der mit $f_S = 64$ kHz abgetastet wird. Die Impulsantwort ist daher eine Abfolge von $L$ Diracsten mit gleicher Amplitude. Mit dem so definierten System filtern wir anschlieend Rauschsignale.

In[]:
fig, ax = plt.subplots(**size)
t = np.arange(-100, 350, 0.1)
ax.plot(t, np.where((t >= 0) & (t < 250), 1, 0), 'r', lw=2)

ax.set_xlabel(r"$t \; / \;\mu\mathrm{s} \;\rightarrow$")

Dieses System wird Moving Average (MA) Filter (gleitender Mittelwert) genannt, das folgende Bild zeigt den Signalflussgraphen eines MA-Filters der Ordnung $N=4$ bzw. Lnge $L=5$. Die Lnge der Impulsantwort entspricht der Anzahl der Taps.

2.1 Einschub: Abschnittsweise definierte Funktionen in Python

Allgemein knnen Sie Funktionen abschnittsweise definieren mit:

2.1.1 List Comprehension

Mit einer sogenannten "List Comprehension" knnen Sie sehr kompakt eine Liste definieren oder modifzieren. Mit y=[i*i for i in t] definieren Sie eine Liste mit einer quadratischen Funktion ber dem Zeitvektor $t$, mit y=[i < T_0 for i in t] eine Liste mit einer Funktion, die 1 ist fr $t < T_0$ und ansonsten 0. Dabei wird das schlampige Typecasting von Python genutzt (True -> 1, False -> 0).

List comprehensions hneln der mathematischen Schreibweise:

$$y = t^2 \text{ fr }t \in \mathbb{R} $$ bzw.

$$y = 1 \text{ fr }t < T_0, 0 \text{ sonst.} $$

Mit einer zustzlichen Bedingung kann man auch kompliziertere Funktionen erzeugen wie die Menge der Quadratzahlen aller geraden Zahlen zwischen 10 und 20: m = [x**2 for x in range(10,21) if x%2 == 0]. Mathematisch gesehen:

$$ m=\{x^{2}\in \mathbb {N} \,|\,10\leq x\leq 20,\,x \bmod 2 = 0\}$$

Normalverteiltes Rauschen:

Normalverteiltes Rauschen, auch als Gausches Rauschen bekannt, ist durch die Normalverteilung (Gausche Verteilung) charakterisiert. In einem normalverteilten Rauschsignal sind die Werte um einen Mittelwert herum symmetrisch verteilt. Die Wahrscheinlichkeitsdichte fr die Signalwerte folgt der Gauschen Glockenkurve. Charakteristische Eigenschaften sind die Schwerpunktsymmetrie um den Mittelwert und die Beschreibung durch den Mittelwert und die Standardabweichung. Viele natrliche Phnomene, wie Messungen von physikalischen Gren oder Umwelteinflsse, knnen normalverteiltes Rauschen aufweisen. Gleichverteiltes Rauschen:

Gleichverteiltes Rauschen, auch als Rechteckverteilung bekannt, ist durch eine konstante Wahrscheinlichkeitsdichte ber einem bestimmten Intervall gekennzeichnet. Alle Werte innerhalb des Intervalls haben die gleiche Wahrscheinlichkeit, aufzutreten. Im Gegensatz zur Normalverteilung gibt es bei der Gleichverteilung keine strkere Wahrscheinlichkeit fr Werte in der Nhe eines Mittelwerts. Jeder Wert innerhalb des Intervalls ist gleich wahrscheinlich. Gleichverteiltes Rauschen wird manchmal als "weies Rauschen" bezeichnet, insbesondere wenn es ber alle Frequenzen gleichmig verteilt ist.

Eine hhere Abtastrate fhrt zu einer feineren zeitlichen Auflsung des Signals, whrend eine niedrigere Abtastrate zu einer grberen zeitlichen Auflsung fhrt. Feinere Details sind hrbar oder eben nicht und dies kann "gedmpfter" klingen

3. Transferfunktion von LTI-Systemen

Zeitdiskrete Systeme werden in Python und Matlab meistens ber die Koeffizienten $a_i$ des rekursiven Teils und die Koeffizienten $b_i$ des nicht-rekursiven Teils angegeben:

$$ H(z) = \frac {Y(z)} {X(z)} = \frac{b_0 + b_1 z^{-1} + b_2 z^{-2} + b_3 z^{-3}}{1 + a_1 z^{-1} + a_2 z^{-2} + a_3 z^{-3}} $$
